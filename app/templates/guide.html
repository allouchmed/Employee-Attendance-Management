Tech Stack

This project containerized by Docker and using Django as a backend.

Database is PostgreSQL and configured inside Docker conatiner.

The project developed by using Test-Driven Development (TDD), therefore, the tests automated by Travis CI and all tests are passing successfully.

The proxy server handles by Nginx and deployed on Digital Ocean Cloud.

The design of page using Bootstrap framework.

Leaves

Once you logged in you'll see Leaves module. There are three main components in this page.

  • Leave Forum
  • Percentage Conatiner
  • Leaves

The forum is using AJAX to make POST request and django backend catches the information to create new object in databse.

The percentage container calculates the available leave days of user. The annual leave limit is 25 by default and automatically defined when new user instance created. But it will handled by supervisor in future development. Try to add some leaves to see the changes. Check following functionalities:

  • Try to send empty forum.
  • Try to provide wrong inputs.
  • Try to request more than 25 days leave.
  • Try to make request in past. Make first date input greater than second date input. (e.g. I want leave from 18-04-2020 to 14-04-2020)

Finally, all leaves will displayed in table. You can delete particular leave if you want. I will also add archive functionality to the leaves. >> Go to Leaves Page
